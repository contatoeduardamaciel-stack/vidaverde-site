# Projeto VidaVerde

Resumo das entregas do projeto VidaVerde, aplicação web sobre preservação ambiental e voluntariado.

- *Entrega I:* Estrutura inicial com HTML e CSS, páginas estáticas e layout responsivo.  
- *Entrega II:* Interatividade básica com menu hambúrguer, cards de projetos e tags.  
- *Entrega III:* JavaScript avançado para formulários, SPA simples e fechamento automático do menu mobile.  
- *Entrega IV:* Preparação para produção, acessibilidade (WCAG 2.1), minificação de arquivos e otimizações.

Cada entrega está organizada em pastas ou versões separadas, mostrando a evolução do projeto.
